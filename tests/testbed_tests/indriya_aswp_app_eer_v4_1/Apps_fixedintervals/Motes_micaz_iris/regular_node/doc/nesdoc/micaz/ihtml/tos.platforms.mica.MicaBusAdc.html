Interface: tos.platforms.mica.MicaBusAdc

interface MicaBusAdc

A simplistic beginning to providing a standard interface to the mica-family 51-pin bus. Just provides the PW0-PW7 digital I/O pins.

Commands
command uint8_t getChannel() Return the A/D channel number to use for one of the ADCn pins.
