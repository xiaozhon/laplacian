Component: tos.chips.atm128.timer.HplAtm128Timer0AsyncP

module HplAtm128Timer0AsyncP

HPL interface to Atmega128 timer 0 in ASYNC mode. This is a specialised HPL component that assumes that timer 0 is used in ASYNC mode and includes some workarounds for some of the weirdnesses (delayed overflow interrupt) of that mode.
