Interface: tos.chips.atm128.adc.Atm128AdcConfig

interface Atm128AdcConfig

Clients of the higher-level A/D system must implement this interface to specify which channel to sample, and with what parameters.

Author:
David Gay

Commands
command uint8_t getChannel() Obtain channel.
command uint8_t getPrescaler() Obtain prescaler value.
command uint8_t getRefVoltage() Obtain reference voltage

Commands - Details

getChannel

command uint8_t getChannel()

Obtain channel.

Returns:
The A/D channel to use. Must be one of the ATM128_ADC_SNGL_xxx or ATM128_ADC_DIFF_xxx values from Atm128Adc.h.

getPrescaler

command uint8_t getPrescaler()

Obtain prescaler value.

Returns:
The prescaler value to use. Must be one of the ATM128_ADC_PRESCALE_xxx values from Atm128Adc.h.

getRefVoltage

command uint8_t getRefVoltage()

Obtain reference voltage

Returns:
The reference voltage to use. Must be one of the ATM128_ADC_VREF_xxx values from Atm128Adc.h.
